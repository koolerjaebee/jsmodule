# templateGenerator: Shiny Gadget for global/app.R template.

Opens a Shiny app that allows users to generate a Shiny project
template.

## Usage

``` r
templateGenerator()
```

## Value

Shiny Gadget including Data, Label info, Table 1, Regression(linear,
logistic), Basic plots

## Details

Shiny Gadget including Data, Label info, Table 1, Regression(linear,
logistic), Basic plots

## Examples

``` r
if (interactive()) {
  templateGenerator()
}
```
